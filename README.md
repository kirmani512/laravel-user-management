# Laravel User Management System

## Description
A user management system built with Laravel featuring authentication and dynamic user interactions.

## Features
- User registration and login
- Role-based access control
- AJAX-powered interactions
- Secure authentication flow

## Tech Stack
- Laravel
- PHP
- JavaScript (AJAX)
- MySQL

## Setup
1. Clone the repository  
2. Run `composer install`  
3. Configure `.env`  
4. Run migrations
